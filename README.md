### 目录文档说明
* consensus -- go实现各类共识算法

* micro -- go微服务相关工具实战
* type -- go数据结构类型
* web -- web框架实战
* tech -- 知识分类
  * frame -- go实用工具和依赖库

  * basic -- 基础知识点
  * interview -- 面试合集
